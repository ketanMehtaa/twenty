```

       │
       ▼
```

## Navigation Flow

### User Navigation Journey

```
┌─────────────────────────────────────────────────────────────┐
│                    User Navigates Application               │
└─────────────────────────────────────────────────────────────┘

Start: User lands on application
  │
  ├─ Not authenticated?
  │  │
  │  └──► /sign-in
  │      │
  │      ├─ User signs in
  │      │
  │      └──► Redirect to intended page or /objects/companies
  │
  └─ Authenticated?
     │
     ├──► Default: /objects/companies (Companies list)
     │    │
     │    ├─ User clicks "Add Filter"
     │    │  └──► Opens filter dropdown (no navigation)
     │    │
     │    ├─ User clicks on a company
     │    │  └──► /objects/companies/:recordId (Company detail)
     │    │      │
     │    │      ├─ Right drawer opens with activity feed
     │    │      │
     │    │      └─ User clicks "Add Activity"
     │    │         └──► Opens activity modal (no navigation)
     │    │
     │    └─ User clicks "Settings" in sidebar
     │       └──► /settings/profile
     │
     ├──► /objects/people (People list)
     │    └─ Similar navigation as companies
     │
     ├──► /objects/opportunities (Deals/Opportunities)
     │    ├─ Board view (Kanban)
     │    └─ Table view
     │
     ├──► /settings/...
     │    ├─ /settings/profile (User profile)
     │    ├─ /settings/workspace (Workspace settings)
     │    ├─ /settings/integrations (Integrations)
     │    ├─ /settings/data-model (Database schema)
     │    └─ /settings/developers (API & Webhooks)
     │
     └──► Command Menu (Cmd+K)
          ├─ Quick search across all records
          ├─ Quick actions (Create record, etc.)
          └─ Navigation shortcuts
```
